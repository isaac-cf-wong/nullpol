# Citing nullpol

If you have used nullpol in your scientific work, please acknowledge us in your
papers and proposals.

```bibtex
@software{nullpol,
  title = {nullpol: Model-independent polarization test of gravitational-wave signals},
  author = {Wong, Isaac C. F. and Ng, Thomas and Cirok, Bal{\'a}zs},
  url = {https://github.com/isaac-cf-wong/nullpol},
  year = {2025}
}
```

Citation metadata is also available in
[`CITATION.cff`](https://github.com/isaac-cf-wong/nullpol/blob/main/CITATION.cff).
